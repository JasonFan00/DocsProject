The Docs Project, 2020

Docs is a spring MVC + thymeleaf web app that can generate a fully navigable, populated, and self-updating website from a mere collection of .md and .txt files organized in a folder hierarchy. For the full overview, please visit the repo linked below. This implementation of Docs serves as a basic resource for CS students. The content of this site is hosted in the repo linked below, which is what Docs uses to generate the site.

Interested in contributing? Please visit the links below and reach out to us!
